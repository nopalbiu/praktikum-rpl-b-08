# WearWoreWorn
Aplikasi katalog fashion untuk menampilkan koleksi outfit terbaru secara interaktif. Untuk kalangan remaja dan dewasa yang sedang mencari outfit terbaru.

1. Katalog Produk: Menampilkan daftar koleksi outfit secara visual.
2. Detail Produk: Informasi bahan, size chart, dan harga.
3. Kategori Produk: Filter berdasarkan jenis pakaian.
